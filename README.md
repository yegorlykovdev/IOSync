# IOSync

Desktop application built with Tauri 2.0, React, TypeScript, and Vite.

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Tauri 2.0 (Rust)
- **Styling:** Tailwind CSS + shadcn/ui
- **Database:** SQLite (via tauri-plugin-sql)
- **Routing:** React Router

## Development

```bash
pnpm install
pnpm tauri dev
```

## Build

```bash
pnpm tauri build
```
